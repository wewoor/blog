我们开源了一个轻量的 Web IDE UI 框架 - Molecule

简介

Molecule 是一个受 VSCode 启发，使用 React.js 构建的 Web IDE UI 框架。通过设计类似 VSCode 扩展机制（Extension），可快速、轻松搭建一个高度抽象的 IDE Workbench UI 系统。

Molecule 具有较高的 Workbench 自定义能力，可帮助需求 IDE UI 业务场景的开发者，实现业务代码和 IDE UI 组件解耦，使业务迭代和 IDE UI 交互迭代可异步进行，降低升级维护成本。同时它基于 Monaco Editor，集成 QuickAccess 和 Keybinding 等功能， 可进行简单的调用 API 封装。

Molecule 可以让你的 Web IDE 产品轻松具备一种可持续进化能力

动机

DTInsight 是一款一站式数据中台 PasS 产品，其中涉及到很多 Web IDE 交互设计。

早期产品界面简单，负载量低，但随着业务发展，产品不断迭代，Workbench 中融入的功能点变得十分密集、复杂。

而架构臃肿、维护成本高、重构升级困难、设计周期长等问题也逐渐暴露出来。技术团队也一直在思考该如何对 Web IDE 交互进行持续优化升级。

2019 年，我们团队参考了市面上做的比较好的 Web IDE 产品，如 Cloud9 IDE、VSCode、Eclipse Theia 等。虽然这些产品很优秀，但与 Workbench 中急需解决的问题有所偏差，于是开始准备自己的 Molecule 开发，试图寻找出一种平衡方案，解决上述问题，并为 Web IDE 带来可持续进化能力。

Molecule 的特点

与其他开源的 Web IDE 的区别？

VSCode 虽然是一款成熟的跨端 IDE 产品，但数栈产品目前主要还是侧重于 IDE UI 偏交互，如果使用 VSCode, 迁移成本高，且 UI 自定义不够灵活，对数栈来说还是太重。
